GenomeFlow is an end-to-end genomics ingestion and quality control platform.

It ingests FASTQ/VCF sequencing data, validates metadata, runs automated QC pipelines, stores results in structured databases and exposes APIs for downstream research and machine learning.

Built as a learning platform for genomics data engineering and bioinformatics infrastructure.
